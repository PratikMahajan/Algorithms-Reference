## Get a Separator in a rotated sorted Array
**Modification of Binary search**


```python
class SortedSeparator(object):
    def getSeparator(self,nums):
        lo=0
        hi=int(len(nums)-1)
        while lo<hi:
            mid=int((lo+hi)/2)
            if nums[mid]>nums[hi]:
                lo=mid+1
            else:
                hi=mid   
        return lo
print(SortedSeparator().getSeparator([7,8,9,10,12,23,45,67,1,3,4,5,6]))
```
